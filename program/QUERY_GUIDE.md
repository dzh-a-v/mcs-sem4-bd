# Query Guide

Available versions:

- English: [QUERY_GUIDE_EN.md](./QUERY_GUIDE_EN.md)
- Russian: [QUERY_GUIDE_RU.md](./QUERY_GUIDE_RU.md)

The two files contain the same structure:

- how to connect to the small and large databases
- how to create and load the small and large databases
- useful `psql` commands
- simple queries
- joins, grouping, and aggregation
- complicated queries with many conditions
- examples with `EXISTS`, `NOT EXISTS`, `WITH`, and window functions
- CSV export examples
